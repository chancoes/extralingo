EXTRALINGO

Project Description

Extralingo is a conversaton-based language learning application designed to be used in supplement with other language learning methods. Taking an immersion learning philosophy, Extralingo seeks to provide a partner and tutor for language learners to converse with utilizing Artifical Intelligence to create realistic sounding conversations which can adapt to the users' needs and responses while providing real-time custom feedback specific to each exchange.

Below are Samples of How Extralingo Works

If the user already has an account with Extralingo, they can sign into the product

The user needs to register for an account with the application to use the product

After logging into the system, the user is greeted with 4 topics of conversation: Food, Weather, Lodging, and Transportation

After the user has selected a lesson, they will be prompted with instructions and a question to begin their conversation. All follow-up conversation will be done in Spanish.
